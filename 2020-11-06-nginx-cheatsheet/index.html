# 安装homebrew

homebrew是一个包类似node中npm, 可以帮助你安装macOS或者linux上没有的软件, 可以使用下列命令进行安装:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
1

# brew services

brew services命令用来快速启动各种服务

brew services start brew services restart brew services stop brew services list

# 安装nginx

brew install nginx
1

查看是否安装成功

nginx -v
1

帮助

nginx -h
1

# 启动

# 快速启动

brew services start nginx
1

# 查看实例

brew services list
1

看到有nginx started就是成功了

# 停止,重新加载配置

  • nginx -t signal
  • 快速关机 nginx -s stop, nginx -s quit
  • nginx -s reload 重新加载配置文件
  • nginx -s reopen 重新打开log 文件

# 查看nginx配置文件

配置文件在以下几个位置之一

/usr/local/nginx/conf, /etc/nginx或者 /usr/local/etc/nginx

可以使用下列命令查看

nginx -t
1
# 配置文件的结构
    http {
        server {
            port 80;
            server_name 'www.chenxiaoyao.cn';
            location / {
                root /data/www/;  //http://localhost/some/example.html => /data/www/some/example.html
            }
            location /images/ {
                root /data;
            }
            location ~\.(gif|jpg|png)$ {
                root /data/images //匹配所有符合的图片
            }
            location / {
                proxy_pass http://chenxiaoyao.cn:3000/;
            }
              location /api/ {
                rewrite  ^/api/(.*)$  /$1 break;
                proxy_pass http://localhost:1234;
                proxy_set_header Host $host;
                proxy_set_header X-Real-IP $remote_addr;
             }
        }
    }
